operator是对k8s的一种扩展，通过自定义资源CRD，实现将应用

  1. kubebuilder init –domain 初始化一个工程
  2. kubebuilder create api –group xxx –verion xxx –kind Xxxx 创建具体的资源
  3. 修改type，定义自定义资源
  4. 修改controller，实现reconcile
